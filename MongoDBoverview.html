Install MongoDB Community Server on Mac

Homebrew will make installation easier on mac. Install homebrew by running this command at Terminal /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh) Use this link to install MongoDB Server using homebrew: https://docs.mongodb.com/manual/tutorial/install

mongodb-on-os-x/

In Terminal, type: brew tap mongodb/brew and hit Enter After completion, type in Terminal: brew install mongodb-community@4.2 and hit Enter Program installed at usr/local/Cellar/mongodb-community In terminal, type: cd usr/local/cellar/mongodb-community and hit Enter In terminal, type: mkdir data and hit Enter In terminal, type: cd data and hit Enter In terminal, type mkdir db and hit Enter In terminal, type /usr/local/etc/mongod.conf and hit Enter to start MongoD server
